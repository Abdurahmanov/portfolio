# Personal Portfolio Website

This is a personal portfolio website built with Vue 3, TypeScript, and Vite. It features a modern design, dark/light mode, and multi-language support (English and Russian).

## How to run

First, install the required dependencies:

```bash
npm install
```

To start the local development server:

```bash
npm run dev
```

Then open `http://localhost:5173/` in your browser.

## How to build

To build the project for production, run:

```bash
npm run build
```

This will create an optimized build in the `dist` folder.

## How to customize

- Edit your personal information, experience, and skills in `src/stores/portfolio.ts`.
- Edit text translations in `src/i18n/index.ts`.
